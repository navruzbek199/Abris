LastPass компании LogMeIn обеспечивает

расширенные возможности аутентификации для бизнеса

Новость. 17.03.2021

Использование многофакторной аутентификации (MFA) при доступе к важным или конфиденциальным веб-ресурсам считается передовой практикой для частных лиц и предприятий, стремящихся обеспечить защиту от киберугроз. LogMeIn, Inc., ведущий поставщик облачных решений для удаленной работы, обеспечивающих возможность работы из любого места, представила улучшенный опыт многофакторной аутентификации с выпуском усовершенствованного мобильного приложения LastPass Authenticator, новой интеграции с ведущими поставщиками

VPN Cisco, Palo Alto Networks и OpenVPN, а также новый интерфейс администрирования MFA. С правильным решением MFA сотрудники могут без проблем войти в систему, а надежные протоколы безопасности предотвращают кибератаки. Недавно выпущенное приложение Authenticator предоставит всем пользователям LastPass — от частных лиц до предприятий — одно комплексное приложение для удовлетворения их потребностей в безопасности.

Недавние исследования показали, что 62 % руководителей ИТ-служб считают, что многофакторная аутентификация (MFA) является наиболее эффективным способом защиты их удаленных сотрудников.

Единое приложение

Для аутенфикации

Бесплатное мобильное приложение, доступное для всех пользователей LastPass, LastPass Authenticator обеспечивает второй уровень аутентификации при входе в хранилище LastPass, сторонние сайты и бизнес-приложения. Этот второй уровень аутентификации предоставляет пользователям возможность подтвердить свою личность перед доступом

к сохраненным паролям в их хранилище LastPass. С этим обновлением LastPass Authenticator будет использовать обновленный пользовательский интерфейс, который теперь включает функцию поиска для уменьшения сложности пользователя и оптимизации процесса аутентификации.

Новые возможности

для бизнеса

Компании могут приобрести дополнительные функции аутентификации с помощью пакета LastPass MFA, который добавляет дополнительный уровень безопасности для их VPN, рабочих станций, поставщиков удостоверений или облачных приложений. Чтобы дать администраторам возможность настраивать MFA в соответствии

с потребностями своей организации, LastPass объединяет отдельное мобильное приложение LastPass MFA в улучшенное мобильное приложение Authenticator. В результате администраторам теперь потребуется только один вариант аутентификации, чтобы удовлетворить ряд требований безопасности своей организации.

Новые функции включают:

• Одно комплексное приложение для аутентификации: одно приложение LastPass Authenticator устраняет путаницу и упрощает контроль для администраторов и конечных пользователей. От простого push-уведомления до защиты без пароля, LastPass Authenticator — это единое место для многофакторных входов конечных пользователей в хранилище и приложения SSO с новыми возможностями поиска для оптимизации безопасного доступа. Это улучшенное приложение в настоящее время развертывается для клиентов. • Интеграция VPN с ведущими поставщиками. Помимо защиты бизнес-приложений, администраторы теперь могут добавлять в свои VPN многофакторную аутентификацию, чтобы обеспечить дополнительный уровень безопасности при доступе сотрудников к сети. Благодаря новой интеграции с ведущими поставщиками VPN, такими как Cisco, Palo Alto Networks, OpenVPN и др., LastPass усиливает безопасность конечных точек, чтобы снизить риск получения доступа к сети киберпреступниками. Полный список поддерживаемых производителей VPN можно найти здесь. • Удобство администрирования MFA: теперь администраторы могут добавить к своим конечным точкам защиту с многофакторной аутентификацией с помощью нового интерфейса администрирования, который повышает прозрачность и упрощает развертывание MFA. Эта функция будет доступна позже во втором квартале.

Когда сотрудники получают доступ к информации компании за пределами традиционного корпоративного периметра, для ИТ-отдела критически важно убедиться, что входящий в систему сотрудник соответствует тому, кем они являются. Благодаря недавно усовершенствованному приложению LastPass Authenticator, новой интеграции с ведущими поставщиками VPN и обновленному интерфейсу администрирования MFA администраторы могут легко выбрать правильный уровень защиты для своей организации, не вызывая путаницы у конечных пользователей.

Дэн ДеМишель, вице-президент LastPass по управлению продуктами в LogMeIn
